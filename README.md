# ToDo-Backend

## Getting Started

1. Clone the repository.
2. Run `npm install` to install the dependencies.
3. Configure the `.env` file with refrence to `.example.env`
4. Run `npm start` to start the server.

### Prerequisites

- Node.js (v14 or higher)
- npm (v7 or higher)
- MongoDB
